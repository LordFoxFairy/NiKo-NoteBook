## 3.1 提示词结构方法论

### 黄金结构公式

```
完整提示词 = 质量词 + 主体描述 + 细节补充 + 风格控制 + 技术参数
```

#### 结构拆解

```
第1层: 质量词（必须）
━━━━━━━━━━━━━━━━━━━━━━━━━━━
masterpiece, best quality, ultra detailed, 8k

第2层: 主体描述（核心）
━━━━━━━━━━━━━━━━━━━━━━━━━━━
1 girl, fitness model, athletic body, gym

第3层: 细节补充（关键）
━━━━━━━━━━━━━━━━━━━━━━━━━━━
long hair, ponytail, sports bra, yoga pants
toned abs, muscular legs, sweat

第4层: 环境与氛围
━━━━━━━━━━━━━━━━━━━━━━━━━━━
modern gym, equipment background, bright lighting
depth of field, bokeh

第5层: 风格控制（可选）
━━━━━━━━━━━━━━━━━━━━━━━━━━━
photorealistic, professional photography
cinematic lighting, highly detailed

第6层: 负面提示词（重要）
━━━━━━━━━━━━━━━━━━━━━━━━━━━
Negative:
(deformed, ugly, bad anatomy, extra limbs:1.4)
(low quality, blurry, watermark:1.2)
```

## 3.4 权重语法精通

### 基础语法

```
语法格式:
(关键词:权重值)

权重范围:
0.5  = 弱化50%
1.0  = 标准（默认）
1.2  = 强化20%
1.5  = 强化50%
2.0  = 强化100%（不推荐过高）
```

### 实战示例

#### 示例1: 强调腹肌

```
普通写法:
toned abs

强化写法:
(toned abs:1.3)

效果对比:
1.0: 隐约可见的腹肌线条
1.3: 明显的马甲线/腹肌
1.5: 非常明显的肌肉线条（可能过度）
```

#### 示例2: 平衡多个要素

```
需求: 既要肌肉线条，又要保持女性柔美

提示词设计:
(athletic body:1.3),        # 强调健美身材
(toned abs:1.2),            # 适度腹肌
(feminine features:1.1),    # 保持女性特征
(soft skin:1.1),            # 柔和肌肤
(muscular but not bulky:1.2) # 有肌肉但不壮硕
```

#### 示例3: 负面权重

```
强力避免畸形:
Negative:
(deformed:1.5),         # 强力避免
(bad anatomy:1.5),      # 强力避免
(extra limbs:1.4),      # 重点避免
(blurry:1.2),           # 一般避免
watermark               # 标准避免
```

## 3.6 不同模型的提示词差异

### SDXL 系列

#### 特点
- 理解能力强
- 支持自然语言
- 可以写长提示词

#### 推荐写法

```
# SDXL风格（接近自然语言）
A professional photograph of a beautiful asian fitness model
with athletic body and toned abs,
wearing a sports bra and yoga pants,
standing in a modern gym with bright lighting,
photorealistic, highly detailed, 8k resolution

# 也可以用标签风格
masterpiece, best quality, 8k,
1 girl, fitness model, athletic body, toned abs,
sports bra, yoga pants, gym background,
professional photography, cinematic lighting
```

---

### Flux.1 系列

#### 特点
- **最强自然语言理解**
- 推荐用句子描述
- 不需要质量词

#### 推荐写法

```
# Flux.1 风格（自然描述）
A stunning photograph of a 25-year-old Asian fitness model
in her prime athletic condition,
showcasing her well-defined abdominal muscles and toned physique.
She's wearing a sleek black sports bra and matching yoga pants,
standing confidently in a modern gym with professional lighting.
The image captures every detail with exceptional clarity,
from the texture of her skin to the subtle sweat glow.

# 简洁版也可以
Athletic Asian woman working out in gym,
professional fitness photography,
highly detailed, photorealistic

⚠️ 不需要: masterpiece, best quality等质量词
```

---

### Midjourney V6

#### 特点
- 艺术性强
- 需要具体描述
- 支持参数控制

#### 推荐写法

```
# Midjourney风格
fitness model asian woman, athletic body,
toned abs, sports bra, yoga pants,
modern gym environment, professional photography,
cinematic lighting, highly detailed --ar 2:3 --v 6 --style raw

# 参数说明
--ar 2:3     : 纵向比例
--v 6        : 版本6
--style raw  : 真实风格（不要艺术化）
--s 200      : 风格化强度（0-1000）
```

---

### DALL-E 3

#### 特点
- 完全自然语言
- 自动优化提示词
- 不支持负面提示词

#### 推荐写法

```
# DALL-E 3 风格（简洁自然）
A professional photograph of an athletic Asian woman
exercising in a modern gym,
wearing sportswear,
with natural lighting and high detail

⚠️ 注意:
- 不要写太长
- 不需要质量词
- 没有Negative Prompt
- DALL-E会自动润色你的提示词
```

## 22.2 核心功能

### 22.2.1 基础生成

```
Web: pika.art

Image-to-Video:
1. 上传图片
2. 输入运动提示 (可选)
3. 调整参数:
   - Motion: 1-4 (运动强度)
   - Camera: Pan/Zoom等
4. Generate

提示词示例:
"Camera slowly zooms out, athlete begins to move"
```

### 22.2.2 特色功能

**1. Expand Canvas (画布扩展)**
```
功能: 扩展视频画面范围
用途: 16:9 → 9:16 (横转竖)

操作:
1. 生成初始视频
2. 点击Expand
3. 选择扩展方向
4. 生成
```

**2. Modify Region (区域修改)**
```
类似ControlNet Inpaint
选择区域 → 修改该部分 → 其余不变

案例:
原视频: 健身房场景
选中背景墙 → "add motivational poster"
→ 仅墙面变化,人物/器材不变
```

**3. Effects (创意效果)**
```
特效库:
- Explode: 爆炸效果
- Melt: 融化
- Inflate: 膨胀
- Cake-ify: 变成蛋糕质感
- Squish: 挤压

适用: 创意视频/趣味内容
```

## 22.4 实战技巧

```python
# API调用示例 (伪代码)

import pika_api

client = pika_api.Client(api_key="...")

# 生成视频
video = client.generate_video(
    image="gym_scene.jpg",
    prompt="Camera pans across gym, equipment visible",
    motion=2,  # 1-4,适中运动
    camera_motion="pan_right",
    duration=3  # 3或5秒
)

# 扩展画布
expanded = client.expand_canvas(
    video=video,
    direction="down",  # 向下扩展
    prompt="reveal more of the gym floor"
)

# 下载
client.download(expanded, "output.mp4")
```

## 22.5 最佳实践

**提示词建议**:
```
✅ "Slow camera zoom on athlete"
✅ "Gentle movement, breathing visible"
✅ "Dynamic pan revealing gym equipment"

❌ "Lots of movement" (太模糊)
❌ "Person jumps and spins" (Pika复杂动作弱)
```

**Motion参数**:
- Motion 1: 微动作 (呼吸/眨眼)
- Motion 2: 标准 (推荐)
- Motion 3: 明显运动
- Motion 4: 剧烈运动 (可能不稳定)
